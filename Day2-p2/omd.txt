<nav class="main-navigation">

</nav>

3. Name at least five block-level elements.
Ans: <address> , <article> , <aside> , <div> , <section> , <nav>

4. When choosing between a <section> and a <div> to represent a section in the page, which one would you prefer?
Ans: <section>

5. Suppose the selector a[href] .c.d[type=text] is given. Which is the key selector?
Ans: .c.d[type=text]

1. What is the width of the content area of a border box element having a 300px width, 30px padding, and 5px border?
Ans: 300px +(2*30px)+(2*5px)=370px

2. What is the width of the left margin of a border box element having a 300px width, 30px padding, auto margin, and 5px border, assuming the viewport width is 540px?
Ans: 540px-(2*5px)-(2*30px)=470px

3. Knowing the properties of the content-box and the border-box model, what elements would the padding-box contain? a)Padding and content b) Only Content c) Only Padding
Ans: padding and content

4. Which properties do the different box-sizing property values influence assuming that all values are given in pixels? 
a) width b)width and height c) border and padding d) border only
Ans: width and height
